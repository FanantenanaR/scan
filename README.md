# Prise en main
## Prerequis:
### 1. NodeJS (18 LTS)
Vous pouvez telecharger ici.
### 2. Ionic CLI
Executer la commande suivante: 
```NPM 
npm install -g @ionic/cli
```
### 3. Installation des dependencies
Executer la commande suivante dans le dossier:
```
npm install
```

## Run
Pour executer le projet:
```
ionic serve
```
